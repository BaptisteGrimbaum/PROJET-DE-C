--------------------------- Projet C Partie 2 ---------------------------

Pour comprendre comment fonctionne le code, il vous suffit de lancer le main :

Choisissez le nombre de niveaux pour la liste chaînée simple. (supérieur à 0)
Choisissez d'afficher ou non la liste. (0 : non ; 1 : oui) 
Choisissez une valeur à rechercher dans la liste.
Le programme affichera si la valeur est présente dans la liste par recherche au niveau 0 et par recherche dichotomique.
Choisissez si vous souhaitez comparer les temps de recherche pour différentes quantités de valeurs entre la recherche au niveau 0 et la recherche dichotomique. (0 : non ; 1 : oui)  
Le programme affichera les temps de recherche pour 1000, 10 000 et 100 000 valeurs.

Notes :
Le programme utilise des saisies sécurisées pour garantir des entrées valides.
Les temps de recherche sont affichés en millisecondes.
